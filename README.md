# 🌌 Nova
Программа для обхода всех видов блокировок сайтов в России. В начале апреля будет крупное обновление.

# <img width="362" height="302" alt="image" src="https://github.com/user-attachments/assets/59ec654b-0448-4624-9b32-1ef901b95b22" />


Все нейросети должны открываться и работать, как и Spotify/Soundcloud и многие другие сайты. Если есть пожелания, говорите и добавлю ваши сайты для всех.

Вотсап и Телеграм заработают без задержек, Дискорд - пока только через winws стратегию. Голос - напрямую (через winws).

Если что-то не работает в одном браузере, пробуйте в другом - иногда мешает старый кэш/куки браузера или его расширения для обхода блокировок - их надо отключить.

Twitch также будет работать в качестве 1080p (а для некоторых каналов доступно 1440p после авторизации в аккаунте). Однако с приходом высокого качества так же может вернуться и реклама - режем её с помощью [Adguard расширения](https://chromewebstore.google.com/detail/adguard-%D0%B0%D0%BD%D1%82%D0%B8%D0%B1%D0%B0%D0%BD%D0%BD%D0%B5%D1%80/bgnkhhnnamicmpeenaelnjfhikgbkllg?hl=ru).

Имейте ввиду, что программа не залезает в настройки системного `C:\Windows\System32\drivers\etc\hosts` - поэтому если там есть обход блокировок, он может конфликтовать с тем, что даёт Nova (рекомендую очистить файл в исходное состояние). Также не забывайте, что если вы видите плашку "Доступ к сайту ограничен законом РФ" - значит ваши DNS-запросы идут провайдеру - рекомендуется использовать в системе любой публичный DNS - например 8.8.8.8 или его зашифрованный DOH вариант в браузере https://dns.google/dns-query (настройка DNS для Хром здесь: `chrome://settings/security`)
